Beetle Chess Engine

Beetle is a chess engine that uses artificial intelligence to plays against opponents. To select optimal moves, and performs a static and dynamic evaluation of various possible game states. Beetle also employs minimax and alpha-beta pruning to efficiently return the best possible moves. Beetle was developed in Python using the chess.py library.

Team ANDD Database Management System

The Team ANDD Database Management System, implemented entirely in Java, handles SQL query execution on a given tableset. The system supports both unindexed and single-indexed tables, and features multiple optimizations to minimize execution time. I completed this project with a group of three other classmates for my Practicum in Database Systems class. Along with writing the code in collaboration, I also was responsible for designing the project architecture, delegating necessary tasks amongst the group, and managing version control. We used JSqlParser to handle SQL queries and JUnit for integration testing.

Retrieval-Augmented Generation Pipeline

The Retrieval-Augmented Generation Pipeline enhances the quality of LLM outputs by enabling the model to reference given documents. The pipeline uses a FAISS vector index and L2 similarity search to identify relevant context and the bge-base-en-v1.5 enmbedding model to generate vector embeddings. These embeddings are then added to the vector index, so that when a text prompt is inputted, the pipeline can reference relevant context and augment the prompt to include these text strings. I completed this as a final project for my Systems Programming class. I developed the pipeline entirely in Python, using the faiss and llama-cpp-python libraries.

Big Red Blockchain

Big Red Blockchain is a proof-of-concept blockchain application written entirely in the OCaml programming language. Users can mine blocks, create wallets and make transactions, and observe the ledger. I worked with a team of 3 of my classmates to develop Big Red Blockchain for our Data Structures and Functional Programming class. My main contributions were implementing wallet functionality and designing test suites. We used Dune for the build system and OUnit for testing.
