Projects

Beetle Chess Engine

Beetle is an engine that uses search trees and move evaluation to play optimal moves against its opponent. It employs the minimax algorithm and alpha-beta pruning to efficiently select moves, which enables nuanced and forward-thinking strategy. Beetle also uses the UCI interface to support play through Chess GUIs and applications like Lichess. Beetle is made in Python and uses the chess.py library.

Team ANDD Database Management System

The Team ANDD Database Management System, implemented entirely in Java, handles SQL query execution on a given tableset. The system supports both unindexed and single-indexed tables, and features multiple optimizations to minimize execution time. I completed this project with a group of three other classmates for my Practicum in Database Systems class. Along with writing the code in collaboration, I also was responsible for designing the project architecture, delegating necessary tasks amongst the group, and managing version control.

RISC-V Interpreter

The RISC-V Interpreter takes in and evaluates programs written in the RISC-V assembly language. It supports a subset of RISC-V instructions for arithmetic, memory operations, and branch operations. The interpreter simulates memory and RAM using linked lists or hash tables. I developed this interpreter as part of my Computer Organization and Systems Programming class. My contribution was implementing the memory simulation and program evaluation. I wrote the interpreter entirely in the C programming language.

Big Red Blockchain

Big Red Blockchain is a proof-of-concept blockchain application written entirely in the OCaml programming language. Users can mine blocks, create wallets and make transactions, and observe the ledger. I worked with a team of 3 of my classmates to develop Big Red Blockchain for our Data Structures and Functional Programming class. My main contributions were implementing wallet functionality and designing test suites.
